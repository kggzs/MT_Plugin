## 📂 Java 源代码文件

### 核心功能类

| 文件名 | 路径 | 功能描述 |
|--------|------|----------|
| **EncodeDecodeMenu.java** | `src/main/java/com/example/myplugin/` | **编码/解码浮动菜单**<br>• 提供 Base64、Hex、Unicode、URL、ROT13、二进制编解码功能<br>• 哈希计算（MD5、SHA-256、SHA-512）<br>• 时间戳与日期双向转换<br>• 撤销和替换功能<br>• 继承自 `BaseTextEditorFloatingMenu` |
| **AICodeAnalysisToolMenu.java** | `src/main/java/com/example/myplugin/` | **AI 代码分析工具菜单**<br>• 在编辑器顶部「编辑」菜单中添加 AI 分析入口<br>• 对编辑器中的全部代码进行分析<br>• 显示思考过程和实时分析结果<br>• 继承自 `BaseTextEditorToolMenu` |
| **AICodeAnalysisFloatingMenu.java** | `src/main/java/com/example/myplugin/` | **AI 快速分析浮动菜单**<br>• 在选中文本后弹出的浮动菜单中添加 AI 分析入口<br>• 对选中的代码片段进行快速分析<br>• 仅在有文本选中时可见<br>• 继承自 `BaseTextEditorFloatingMenu` |
| **QuickInsertFunction.java** | `src/main/java/com/example/myplugin/` | **快速插入时间功能**<br>• 在光标位置快速插入当前日期时间<br>• 格式：`yyyy-MM-dd HH:mm:ss`<br>• 无需选中文本，随时可用<br>• 继承自 `BaseTextEditorFunction` |
| **MyPreference.java** | `src/main/java/com/example/myplugin/` | **插件偏好设置/关于页面**<br>• 显示插件名称、作者、网站<br>• AI 配置统一入口（API 地址、模型名称、API 密钥）<br>• 提示词独立配置入口<br>• 重置配置功能<br>• 各功能的使用说明<br>• 实现 `PluginPreference` 接口 |

| 文件名 | 路径 | 功能描述 |

## 📝 详细说明

### EncodeDecodeMenu.java

**功能概述**：
编码/解码浮动菜单，是插件中功能最丰富的辅助工具。

**主要功能**：
1. **编码/解码**
   - Base64 编码/解码
   - Hex (十六进制) 编码/解码
   - Unicode 编码/解码
   - URL 编码/解码
   - ROT13 编码/解码
   - 二进制编码/解码

2. **哈希计算**
   - MD5 哈希
   - SHA-256 哈希
   - SHA-512 哈希

3. **时间戳转换**
   - 时间戳转日期
   - 日期转时间戳
   - 支持多种日期格式自动识别

4. **辅助功能**
   - 撤销操作
   - 替换文本
   - 自动选中文本填充

---

### AICodeAnalysisToolMenu.java

**功能概述**：
AI 代码分析工具菜单，提供全文代码分析功能。

**主要功能**：
1. 在编辑器顶部「编辑」菜单中添加「AI 代码分析」入口
2. 对编辑器中的全部代码进行 AI 分析
3. 实时显示思考过程
4. 弹出结果对话框显示分析结果
5. 支持复制分析结果

**使用场景**：
- 完整代码审查
- 全面质量评估
- 深度问题诊断

---

### AICodeAnalysisFloatingMenu.java

**功能概述**：
AI 快速分析浮动菜单，提供选中代码片段的快速诊断。

**主要功能**：
1. 在选中文本后的浮动菜单中添加「AI 快速分析」入口
2. 仅分析选中的代码片段
3. 快速返回分析结果
4. 仅在有文本选中时显示

**使用场景**：
- 特定函数检查
- 代码块诊断
- 可疑代码段分析

---

### AIHelper.java

**功能概述**：
AI 工具类，封装所有 AI 相关的网络请求和配置管理逻辑。

**核心方法**：

| 方法 | 功能 |
|------|------|
| `getApiUrl(context)` | 获取 API 地址 |
| `getAiModel(context)` | 获取 AI 模型名称 |
| `getApiKey(context)` | 获取 API 密钥 |
| `getPrompt(context)` | 获取提示词 |
| `setApiUrl(context, url)` | 保存 API 地址 |
| `setAiModel(context, model)` | 保存 AI 模型名称 |
| `setApiKey(context, key)` | 保存 API 密钥 |
| `setPrompt(context, prompt)` | 保存自定义提示词 |
| `resetToDefault(context)` | 重置为默认配置 |
| `analyzeCodeWithThinking(...)` | AI 分析代码（显示思考过程） |
| `analyzeCodeWithAI(...)` | AI 分析代码（通用方法） |
| `runOnMainThread(action)` | 在主线程执行任务 |

**默认配置**：
- API 地址：`https://api.kggzs.cn/v1`
- 模型名称：`deepseek-v3.2`
- API 密钥：`sk-K1m4b0U2WoorIub7EhbQTIYRFpQhURRXMdIoZBywCEruujOa`
- 提示词：从 `assets/提示词.txt` 读取
