## Java 源代码文件

### 核心功能类

#### [EncodeDecodeMenu.java](../src/main/java/com/kggzs/cn/mt/EncodeDecodeMenu.java)

**功能**: 编码/解码浮动菜单实现

**职责**:
- 实现 `FloatingMenu` 接口，在文本编辑器浮动菜单中添加「编码/解码」选项
- 提供多种编码/解码功能：
  - Base64 编码/解码
  - Hex (十六进制) 编码/解码
  - Unicode 编码/解码
  - URL 编码/解码
  - ROT13 编码/解码
  - 二进制编码/解码
- 提供哈希计算功能（MD5、SHA-256、SHA-512）
- 提供时间戳转换功能

**关键方法**:
- `onClick(View view)` - 处理菜单点击事件
- `showEncodeDecodeDialog()` - 显示编码/解码对话框
- `encodeBase64()` / `decodeBase64()` - Base64 编解码
- `encodeHex()` / `decodeHex()` - Hex 编解码
- `calculateHash(String algorithm)` - 计算哈希值
- `convertTimestamp()` - 时间戳转换

#### [AICodeAnalysisToolMenu.java](../src/main/java/com/kggzs/cn/mt/AICodeAnalysisToolMenu.java)

**功能**: AI 代码分析工具菜单实现

**职责**:
- 实现 `ToolMenu` 接口，在文本编辑器工具栏中添加「AI 代码分析」选项
- 对编辑器中的全部代码进行 AI 智能分析
- 使用 `AICodeAnalysisHelper` 辅助类处理公共逻辑

**关键方法**:
- `onClick(View view)` - 处理菜单点击，启动全文分析
- 使用 `AICodeAnalysisHelper` 显示提示词输入对话框
- 使用 `AICodeAnalysisHelper` 执行 AI 分析流程

---

#### [AICodeAnalysisFloatingMenu.java](../src/main/java/com/kggzs/cn/mt/AICodeAnalysisFloatingMenu.java)

**功能**: AI 快速分析浮动菜单实现

**职责**:
- 实现 `FloatingMenu` 接口，在文本编辑器浮动菜单中添加「AI 快速分析」选项
- 仅分析选中的代码片段，提供快速诊断
- 使用 `AICodeAnalysisHelper` 辅助类处理公共逻辑

**关键方法**:
- `onClick(View view)` - 处理菜单点击，启动选中分析
- 检查是否有选中文本，无选中时提示用户
- 使用 `AICodeAnalysisHelper` 执行 AI 分析流程

---

#### [AICodeAnalysisHelper.java](../src/main/java/com/kggzs/cn/mt/AICodeAnalysisHelper.java)

**功能**: AI 分析辅助类（封装公共逻辑）

**职责**:
- 封装 AI 分析的公共逻辑，消除代码重复
- 显示提示词输入对话框
- 处理快速提示词按钮点击
- 处理 Skill 多选逻辑
- 执行 AI 分析流程（网络请求、结果处理）
- 显示分析结果对话框
- 清理 Markdown 格式，提升纯文本阅读体验
- 处理后台运行模式

**关键方法**:
- `showPromptDialog()` - 显示提示词输入对话框
- `executeAnalysis()` - 执行 AI 分析
- `showResultDialog()` - 显示分析结果
- `cleanMarkdown()` - 清理 Markdown 格式
- `buildRequestBody()` - 构建请求体

#### [QuickInsertFunction.java](../src/main/java/com/kggzs/cn/mt/QuickInsertFunction.java)

**功能**: 快速插入时间功能实现

**职责**:
- 实现 `FloatingMenu` 接口，在浮动菜单中添加「插入时间」选项
- 在光标位置快速插入当前日期时间
- 支持三种显示模式：不带时间、带时间、自定义格式
- 支持多种时间格式（公历/农历/干支）
- 使用 `TimeFormatHelper` 获取格式配置

**关键方法**:
- `onMenuClick()` - 处理菜单点击，插入时间
- `getFormattedTime()` - 获取格式化后的时间

#### [MyPreference.java](../src/main/java/com/kggzs/cn/mt/MyPreference.java)

**功能**: 插件偏好设置界面

**职责**:
- 实现 `Preference` 接口，提供插件设置界面

  - AI 能力配置（提示词、快速提示词、Skill）

- 时间格式设置：
  - 三种显示模式（不带时间/带时间/自定义）
  - 10种预设格式选择
  - 自定义格式编辑
- 功能使用说明

**关键方法**:
- `onCreate()` - 创建设置界面
- `showApiConfigDialog()` - 显示 API 配置对话框
- `showAiCapabilityConfigDialog()` - 显示 AI 能力配置对话框
- `showTimeFormatConfigDialog()` - 显示时间格式设置对话框
- `showCustomFormatEditorDialog()` - 显示自定义格式编辑对话框
- `resetAllConfig()` - 重置所有配置

### 工具类

#### [AIHelper.java](../src/main/java/com/kggzs/cn/mt/util/AIHelper.java)

**功能**: AI 工具类（核心网络逻辑和配置管理）

**职责**:
- 管理 AI API 配置（地址、模型、密钥）
- 管理提示词配置（全局分析、简短分析）
- 管理快速提示词列表
- 管理自定义 Skill 列表
- 执行 AI API 网络请求
- 处理流式响应（SSE）
- 提供默认配置值

**关键方法**:
- `getApiUrl()` / `setApiUrl()` - API 地址管理
- `getModelName()` / `setModelName()` - 模型名称管理
- `getApiKey()` / `setApiKey()` - API 密钥管理
- `getGlobalPrompt()` / `setGlobalPrompt()` - 全局提示词管理
- `getShortPrompt()` / `setShortPrompt()` - 简短提示词管理
- `getQuickPrompts()` / `setQuickPrompts()` - 快速提示词管理
- `getSkills()` / `setSkills()` - Skill 管理
- `executeAnalysis()` - 执行 AI 分析请求
- `resetToDefaults()` - 重置为默认配置

**默认配置**:
- API 地址: `https://api.kggzs.cn/v1`
- 模型名称: `MT-v1`
- API 密钥: `sk-MT-kggzs-API-key`

#### [TimeFormatHelper.java](../src/main/java/com/kggzs/cn/mt/util/TimeFormatHelper.java)

**功能**: 时间格式配置工具

**职责**:
- 管理时间格式配置（公历/农历）
- 管理显示模式（不带时间/带时间/自定义）
- 提供多种时间格式选项
- 支持自定义格式解析
- 格式化当前时间

**显示模式**:

| 模式 | 说明 |
|------|------|
| MODE_DATE_ONLY | 只显示日期，不带时分秒 |
| MODE_DATE_TIME | 显示日期+时分秒 |
| MODE_CUSTOM | 使用自定义格式 |

**预设格式**:

| 格式类型 | 格式名称 | 示例 |
|----------|----------|------|
| 公历 | 标准中文 | 2026年5月20日 |
| 公历 | ISO | 2026-05-20 |
| 公历 | 斜杠 | 2026/5/20 |
| 公历 | 紧凑 | 20260520 |
| 公历 | 带星期 | 2026年5月20日 星期三 |
| 农历 | 传统汉字 | 丙午年四月初四 |
| 农历 | 农历简写 | 农历四月初四 |
| 农历 | 干支纪日 | 丙午年 癸巳月 甲午日 |
| 农历 | 农历+阿拉伯 | 农历2026年四月初四 |
| 农历 | 公农历并列 | 2026-05-20（丙午年四月初四） |

**自定义格式标记**:

| 标记 | 说明 | 示例 |
|------|------|------|
| yyyy | 4位年份 | 2026 |
| yy | 2位年份 | 26 |
| MM | 补0月份 | 05 |
| M | 月份 | 5 |
| N | 农历月份 | 四月 |
| dd | 补0日期 | 20 |
| d | 日期 | 20 |
| e | 农历日期 | 初四 |
| E | 星期 | 周三 |
| a | 时段 | 上午/下午 |
| aa | 精确时段 | 傍晚/凌晨 |
| HH | 24时补0 | 19 |
| H | 24时 | 19 |
| hh | 12时补0 | 07 |
| h | 12时 | 7 |
| mm | 补0分钟 | 08 |
| m | 分钟 | 8 |
| ss | 补0秒数 | 55 |
| s | 秒数 | 55 |
| l | 时辰 | 酉 |

**关键方法**:
- `getTimeMode()` / `setTimeMode()` - 显示模式管理
- `getTimeFormatType()` / `setTimeFormatType()` - 时间格式管理
- `getCustomFormatString()` / `setCustomFormatString()` - 自定义格式管理
- `getFormattedTime()` - 格式化时间
- `parseCustomFormat()` - 解析自定义格式

#### [LunarCalendar.java](../src/main/java/com/kggzs/cn/mt/util/LunarCalendar.java)

**功能**: 农历计算工具

**职责**:
- 公历转农历计算
- 干支纪年/月/日计算
- 农历月份和日期名称转换
- 时辰计算
- 时段计算（上午/下午/傍晚/凌晨等）

**关键方法**:
- `LunarCalendar(Date date)` - 构造函数，自动转换公历到农历
- `getGanZhiYear()` - 获取干支年（如：丙午）
- `getGanZhiMonth()` - 获取干支月（如：癸巳）
- `getGanZhiDay()` - 获取干支日（如：甲午）
- `getFullLunarString()` - 获取完整农历字符串
- `getSimpleLunarString()` - 获取简写农历字符串
- `getGanZhiFullString()` - 获取干支纪日格式
- `getZodiac()` - 获取生肖
- `getShiChen(int hour)` - 获取时辰
- `getDetailedPeriod(int hour)` - 获取精确时段

## 文件依赖关系

```
MyPreference (设置界面)
    ├── AIHelper (配置管理)
    └── TimeFormatHelper (时间格式)

AICodeAnalysisToolMenu (全文分析)
    └── AICodeAnalysisHelper (分析辅助)
            └── AIHelper (网络请求)

AICodeAnalysisFloatingMenu (选中分析)
    └── AICodeAnalysisHelper (分析辅助)
            └── AIHelper (网络请求)

EncodeDecodeMenu (编码解码)
    └── (无依赖，使用标准库)

QuickInsertFunction (插入时间)
    ├── TimeFormatHelper (格式配置)
    └── LunarCalendar (农历计算)
```
