# DEMO 功能演示集

本文档详细演示 MTKang Plugin 的各项功能使用方法。

---

## 一、编码/解码工具演示

### 1.1 Base64 编码/解码

**场景**: 将文本进行 Base64 编码或解码

**步骤**:

1. 在文本编辑器中选中要处理的文本
2. 点击浮动菜单中的「编码/解码」
3. 在弹出的对话框中：
   - 输入框会自动填充选中的文本
   - 点击「Base64编码」按钮进行编码
   - 点击「Base64解码」按钮进行解码

**示例**:

```
原始文本: Hello World
Base64编码: SGVsbG8gV29ybGQ=
Base64解码: Hello World
```

---

### 1.2 Hex (十六进制) 编码/解码

**场景**: 将文本转换为十六进制格式或从十六进制还原

**步骤**:

1. 选中文本后点击「编码/解码」
2. 点击「Hex编码」或「Hex解码」

**示例**:

```
原始文本: Hello
Hex编码: 48656c6c6f
Hex解码: Hello
```

---

### 1.3 Unicode 编码/解码

**场景**: 将文本转换为 Unicode 转义序列或还原

**步骤**:

1. 选中文本后点击「编码/解码」
2. 点击「Unicode编码」或「Unicode解码」

**示例**:

```
原始文本: 你好
Unicode编码: \u4f60\u597d
Unicode解码: 你好
```

---

### 1.4 URL 编码/解码

**场景**: 对 URL 参数进行编码或解码

**步骤**:

1. 选中文本后点击「编码/解码」
2. 点击「URL编码」或「URL解码」

**示例**:

```
原始文本: name=张三&age=20
URL编码: name=%E5%BC%A0%E4%B8%89&age=20
URL解码: name=张三&age=20
```

---

### 1.5 ROT13 编码/解码

**场景**: 使用 ROT13 凯撒密码变体进行文本转换

**步骤**:

1. 选中文本后点击「编码/解码」
2. 点击「ROT13编码」或「ROT13解码」

**示例**:

```
原始文本: Hello World
ROT13编码: Uryyb Jbeyq
ROT13解码: Hello World
```

> 注: ROT13 是自反的，编码和解码操作相同

---

### 1.6 二进制编码/解码

**场景**: 将文本转换为二进制格式或还原

**步骤**:

1. 选中文本后点击「编码/解码」
2. 点击「二进制编码」或「二进制解码」

**示例**:

```
原始文本: AB
二进制编码: 0100000101000010
二进制解码: AB
```

## 二、哈希计算演示

### 2.1 MD5 哈希

**场景**: 计算文本的 MD5 哈希值

**步骤**:

1. 在编码/解码对话框中输入文本
2. 点击「哈希计算」按钮
3. 选择「MD5」
4. 点击「复制」将结果复制到剪贴板

**示例**:

```
输入文本: Hello World
MD5: b10a8db164e0754105b7a99be7e93fe0
```

---

### 2.2 SHA-256 哈希

```
SHA-256: a591a6d40bf420404a011733cfb0b880d4c1f9e6a1e8c8e8e8e8e8e8e8e8e8e8
```

---

### 2.3 SHA-512 哈希

**示例**:

```
输入文本: Hello World
SHA-512: 2c74fd17edafd80e8447b584be3b...（128位十六进制）
```

## 三、时间戳转换演示

### 3.1 时间戳转日期

**场景**: 将数字时间戳转换为可读的日期格式

**步骤**:

1. 在编码/解码对话框中输入时间戳
2. 点击「时间戳转换」

**示例**:

```
输入时间戳: 1716124800
转换结果: 2024-05-19 16:00:00

输入时间戳: 1716124800000 (毫秒)
转换结果: 2024-05-19 16:00:00
```

---

### 3.2 日期转时间戳

**场景**: 将日期字符串转换为时间戳

**步骤**:

1. 在编码/解码对话框中输入日期字符串
2. 点击「时间戳转换」

**支持的格式**:
- `yyyy-MM-dd HH:mm:ss`
- `yyyy/MM/dd HH:mm:ss`
- `yyyy-MM-dd HH:mm`
- `yyyy/MM/dd HH:mm`
- `yyyy-MM-dd`
- `yyyy/MM/dd`

**示例**:

```
输入日期: 2024-05-19 16:00:00
转换结果: 1716124800

输入日期: 2024/05/19
转换结果: 1716076800
```

## 四、AI 代码分析演示

### 4.1 全文分析

**场景**: 对编辑器中的全部代码进行 AI 智能分析

**步骤**:

1. 打开文本编辑器，加载要分析的代码文件
2. 点击顶部「编辑」菜单中的「AI 代码分析」
3. 在弹出的对话框中：
   - 输入自定义提示词（可选）
   - 点击快速提示词按钮追加预设内容（可选）
   - 选择要应用的 Skill（可选）
   - 勾选「后台运行」可让分析在后台进行
4. 点击「开始分析」
5. 等待分析完成，查看结果

**示例流程**:

```
1. 打开 MainActivity.java 文件
2. 点击「AI 代码分析」
3. 输入提示词: "请分析这段代码的性能问题"
4. 点击「开始分析」
5. AI 返回分析结果:
   - 代码功能概述
   - 性能问题分析
   - 优化建议
```

---

### 4.2 选中分析（快速分析）

**场景**: 仅分析选中的代码片段

**步骤**:

1. 在文本编辑器中选中要分析的代码片段
2. 点击浮动菜单中的「AI 快速分析」
3. 在弹出的对话框中设置提示词
4. 点击「开始分析」

**示例**:

```
选中的代码:
public void processData(List<String> data) {
    for (String item : data) {
        System.out.println(item);
    }
}

AI 分析结果:
1. 功能: 遍历列表并打印每个元素
2. 问题: 使用 System.out.println 不适合生产环境
3. 建议: 使用日志框架如 SLF4J
```

---

### 4.3 快速提示词使用

**场景**: 快速追加预设的提示词

**步骤**:

1. 在 AI 分析对话框中
2. 点击「快速提示词」按钮
3. 从列表中选择预设提示词
4. 提示词会自动追加到输入框

**预设提示词示例**:
- "请检查代码中的潜在 bug"
- "请分析代码的性能问题"
- "请提供代码优化建议"
- "请检查代码是否符合最佳实践"

---

### 4.4 自定义 Skill 使用

**场景**: 使用自定义 Skill 增强分析能力

**步骤**:

1. 在插件设置中配置 Skill
2. 在 AI 分析对话框中
3. 点击「Skill 选择」
4. 勾选要应用的 Skill
5. Skill 内容会追加到提示词末尾

**Skill 示例**:
- Java 代码审查专家
- 性能优化顾问
- 安全漏洞检测器

## 六、设置界面演示

### 6.1 API 配置

**场景**: 自定义 AI API 配置

**步骤**:

1. 打开 MT 管理器的插件管理
2. 找到 MTKang Plugin，点击设置
3. 点击「API 配置」
4. 在弹出的对话框中：
   - 修改 API 地址
   - 修改模型名称
   - 修改 API 密钥
5. 点击「保存」

---

### 6.2 AI 能力配置

**场景**: 自定义 AI 分析能力

**步骤**:

1. 在设置界面点击「AI 能力配置」
2. 在弹出的对话框中：
   - 编辑全局分析提示词
   - 编辑简短分析提示词
   - 管理快速提示词（添加/编辑/删除）
   - 管理 Skill（添加/编辑/删除）
3. 点击「保存」

---

### 6.3 时间格式设置

**场景**: 设置默认的时间插入格式

**步骤**:

1. 在设置界面点击「时间格式设置」
2. 选择喜欢的格式：
   - 公历格式选择
   - 农历格式选择
3. 点击「保存」

---

### 6.4 重置配置

**场景**: 恢复所有设置为默认值

**步骤**:

1. 在设置界面点击「重置配置」
2. 确认重置
3. 所有配置恢复为默认值

## 八、常见使用场景

### 场景 1: 分析 Java 代码

```
1. 打开 .java 文件
2. 点击「AI 代码分析」
3. 输入: "请分析这段 Java 代码的设计模式"
4. 查看分析结果
```

### 场景 2: 快速 Base64 解码

```
1. 选中 Base64 编码的文本
2. 点击「编码/解码」
3. 点击「Base64解码」
4. 点击「替换」将结果替换到编辑器
```

### 场景 3: 计算文件校验和

```
1. 复制文件内容到编辑器
2. 全选文本
3. 点击「编码/解码」
4. 点击「哈希计算」
5. 选择 MD5 或 SHA-256
6. 复制哈希值
```

### 场景 4: 时间戳快速查看

```
1. 输入时间戳数字
2. 点击「时间戳转换」
3. 查看对应的日期时间
```
