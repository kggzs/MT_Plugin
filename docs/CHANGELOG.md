# 更新日志

## v2.0.2 (当前版本) - 2026-05-20

### 新增功能

- 🌐 **多语言支持** - 支持10种语言界面
  - 中文（默认）、繁体中文、日语、韩语
  - 阿拉伯语、德语、西班牙语、法语、俄语、英文
- 🔧 **AI 分析后台运行** - 支持后台分析模式，完成后弹出结果
- 🔧 **快速提示词** - 新增快速提示词功能，最多配置10条
- 🔧 **自定义 Skill** - 新增自定义 Skill 功能，可追加到提示词末尾
- 🔧 **时间格式多样化** - 时间插入支持多种格式
  - 公历格式：标准中文、ISO、斜杠、紧凑、带星期
  - 农历格式：传统汉字、农历简写、干支纪日、农历+阿拉伯数字、公农历并列
- 🔧 **赞助功能** - 设置界面支持赞助二维码（微信/支付宝）

### 优化改进

- 🔧 AI 分析辅助类封装，消除重复代码
- 🔧 Markdown 格式清理，提升纯文本阅读体验
- 🔧 取消分析二次确认，防止误操作
- 🔧 API 密钥遮蔽显示，保护隐私

### 技术细节

- 新增 `AICodeAnalysisHelper.java` 封装公共逻辑
- 新增 `TimeFormatHelper.java` 时间格式配置
- 新增 `LunarCalendar.java` 农历计算工具
- 支持10种语言的语言包文件

## v2.0 - 2026-04-01

### 新增功能

- ✨ **编码/解码工具**
  - Base64 编码/解码
  - Hex (十六进制) 编码/解码
  - Unicode 编码/解码
  - URL 编码/解码
  - ROT13 编码/解码
  - 二进制编码/解码
- ✨ **哈希计算** - 支持 MD5、SHA-256、SHA-512
- ✨ **时间戳转换** - 支持时间戳与日期时间双向转换
- ✨ **AI 代码分析**
  - 全文分析功能
  - 选中分析功能
  - 思考过程实时显示
  - 流式输出
- ✨ **快速插入时间** - 在光标位置快速插入当前时间
- ✨ **插件设置界面** - 完善的设置和功能说明

### 技术细节

- 实现 `EncodeDecodeMenu.java` 编码/解码浮动菜单
- 实现 `AICodeAnalysisToolMenu.java` AI 分析工具菜单
- 实现 `AICodeAnalysisFloatingMenu.java` AI 分析浮动菜单
- 实现 `QuickInsertFunction.java` 快速插入时间
- 实现 `MyPreference.java` 插件设置界面
- 实现 `AIHelper.java` AI 网络请求和配置管理

## 升级指南

### 从 v2.0.1 升级到 v2.0.2

1. 下载最新版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 配置会自动保留，无需重新设置

### 从 v2.0 升级到 v2.0.1

1. 下载 v2.0.1 版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 如需使用自定义 AI 配置，请在设置中配置

### 从 v1.0 升级到 v2.0

1. 下载 v2.0 版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 所有新功能立即可用
