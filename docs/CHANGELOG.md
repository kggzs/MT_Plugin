# 更新日志

## v2.1.0 (当前版本) - 2026-06-05

### 新增功能

- 🤖 **AI 对话功能** - 新增与 AI 直接聊天的交互界面
  - 多轮对话支持，AI 会记住上下文
  - 流式响应，实时显示 AI 回复
  - 对话历史管理，可随时清空
  - 停止控制，支持中断正在进行的对话
- 🔌 **MCP 服务配置** - 支持 MCP (Model Context Protocol) 协议
  - MCP 服务器地址配置
  - 连接测试功能
  - 工具列表查看
  - MCP Skill 管理
  - 启用/禁用控制
- 🔧 **MCP Agent 模式** - AI 对话支持自动调用 MCP 工具
  - AI 自动判断是否需要调用工具
  - 系统自动执行 MCP 工具并反馈结果
  - 支持最多 10 轮工具调用循环
  - 支持 tools/list、tools/call、resources/list、resources/read 等标准方法

### 技术细节

- 新增 `AIChatMenu.java` AI 对话浮动菜单
- 新增 `MCPServiceMenu.java` MCP 服务配置菜单
- 新增 `AIChatHelper.java` AI 对话辅助类（支持 MCP Agent 模式）
- 新增 `MCPClient.java` MCP 客户端（基于 JSON-RPC 2.0）
- 新增 MCP 相关语言包键值（约 30 个）
- 注册新接口：`com.kggzs.cn.mt.AIChatMenu`

## v2.0.2 - 2026-05-20

### 新增功能

- 🌐 **多语言支持** - 支持10种语言界面
  - 中文（默认）、繁体中文、日语、韩语
  - 阿拉伯语、德语、西班牙语、法语、俄语、英文
- 🔧 **AI 分析后台运行** - 支持后台分析模式，完成后弹出结果
- 🔧 **快速提示词** - 新增快速提示词功能，最多配置10条
- 🔧 **自定义 Skill** - 新增自定义 Skill 功能，可追加到提示词末尾
- 🔧 **时间格式多样化** - 时间插入支持多种格式
  - 公历格式：标准中文、ISO、斜杠、紧凑、带星期
  - 农历格式：传统汉字、农历简写、干支纪日、农历+阿拉伯数字、公农历并列
- 🔧 **赞助功能** - 设置界面支持赞助二维码（微信/支付宝）

### 优化改进

- 🔧 AI 分析辅助类封装，消除重复代码
- 🔧 Markdown 格式清理，提升纯文本阅读体验
- 🔧 取消分析二次确认，防止误操作
- 🔧 API 密钥遮蔽显示，保护隐私

### 技术细节

- 新增 `AICodeAnalysisHelper.java` 封装公共逻辑
- 新增 `TimeFormatHelper.java` 时间格式配置
- 新增 `LunarCalendar.java` 农历计算工具
- 支持10种语言的语言包文件

## 升级指南

### 从 v2.0.3 升级到 v2.1.0

1. 下载最新版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 配置会自动保留，无需重新设置
4. 新功能：AI 对话、MCP 服务配置立即可用

### 从 v2.0.2 升级到 v2.0.3

1. 下载最新版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 配置会自动保留，无需重新设置

### 从 v2.0 升级到 v2.0.1

1. 下载 v2.0.1 版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 如需使用自定义 AI 配置，请在设置中配置

### 从 v1.0 升级到 v2.0

1. 下载 v2.0 版本的 `.mtp` 文件
2. 在 MT 管理器中安装新版本
3. 所有新功能立即可用
