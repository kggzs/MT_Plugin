> **优化日期**：2026年4月10日

## ✅ 优化内容总结

### 1. 语言包文件重命名（规范化）

| 原文件名 | 新文件名 | 说明 |
|---------|---------|------|
| `strings-en.mtl` | `strings-zh-CN.mtl` | 简体中文语言包（符合 MT 插件开发规范） |

**修改原因**：
- 原命名 `strings-en.mtl` 不符合 MT 语言包查找规则
- 正确命名应为 `strings-zh-CN.mtl`（简体中文）
- 修复后中文用户可以正确加载中文语言包

---

### 2. 新增繁体中文语言包

**文件名**：`strings-zh-TW.mtl`

**语言包覆盖**：
| 语言 | 文件名 | 词条数量 | 状态 |
|------|--------|---------|------|
| 英文（默认） | `strings.mtl` | 193 | ✅ 完整 |
| 简体中文 | `strings-zh-CN.mtl` | 193 | ✅ 完整 |
| 繁体中文 | `strings-zh-TW.mtl` | 193 | ✅ 完整 |

**语言查找优先级**（MT 规范）：
```
strings-zh-CN.mtl → strings-zh.mtl → strings.mtl → MT 内置语言包
```

### 4. 代码硬编码字符串替换

**修改文件**：

| 文件 | 修改行数 | 说明 |
|------|---------|------|
| `AICodeAnalysisToolMenu.java` | 1 行 | name() 方法返回 `{ai_code_analysis}` |
| `AICodeAnalysisHelper.java` | 30+ 行 | 所有硬编码字符串替换为 `{key}` 格式 |
| `MyPreference.java` | 6 行 | 重置配置对话框文本本地化 |

**替换示例**：

```java
// ❌ 替换前（硬编码中文）
pluginUI.showToast("已添加快速提示词");

// ✅ 替换后（使用语言包）
pluginUI.showToast("{ai_prompt_added}");
```

```java
// ❌ 替换前
return "AI代码分析";

// ✅ 替换后
return "{ai_code_analysis}";
```
