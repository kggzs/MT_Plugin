## 一、优化内容

### 1.1 界面结构优化

**优化前**: 单一列表，信息分散

**优化后**: 分组结构，层次清晰

```
插件信息分组
├── 插件名称
├── 作者信息
├── 官方网站
└── 开源地址

支持作者分组
├── 微信收款码
└── 支付宝收款码

AI 代码分析配置分组
├── API 配置
├── AI 能力配置
└── 重置配置

时间格式设置分组
└── 时间格式设置

使用说明分组
├── 编码/解码说明
├── AI 分析说明
└── 时间插入说明
```

### 1.2 功能增强

| 功能 | 说明 |
|------|------|
| API 配置 | 统一配置 API 地址、模型、密钥 |
| AI 能力配置 | 配置提示词、快速提示词、Skill |
| 时间格式设置 | 选择时间显示格式 |
| 赞助功能 | 显示收款码 |
| 重置配置 | 一键恢复默认配置 |

### 1.3 用户体验改进

- 配置项分组清晰
- 操作按钮位置合理
- 说明文字详细
- 支持滚动查看

## 二、实现方式

### 2.1 主设置界面

```java
public class MyPreference implements PluginPreference {
    @Override
    public void onBuild(PluginContext context, Builder builder) {
        builder.title("{plugin_name}")
               .subtitle("{plugin_description}");
        
        // 插件信息分组
        builder.addHeader("{plugin_info}");
        builder.addText("{author}").summary("康哥");
        builder.addText("{website}").url("http://www.kggzs.cn");
        
        // AI 配置分组
        builder.addHeader("{ai_config}");
        builder.addText("{api_config}").onClick(v -> showApiConfigDialog());
        builder.addText("{ai_capability_config}").onClick(v -> showAiCapabilityConfigDialog());
        
        // 其他分组...
    }
}
```

### 2.2 配置对话框

- API 配置对话框 - 统一编辑 API 地址、模型、密钥
- AI 能力配置对话框 - 编辑提示词、管理快速提示词和 Skill
- 时间格式对话框 - 选择时间显示格式
