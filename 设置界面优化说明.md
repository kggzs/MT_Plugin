# 设置界面优化说明

## 更新概述

优化 AI 配置设置界面，将 API 地址、模型名称、API 密钥整合到统一配置页面，提示词单独配置。

## 主要变更

### 1. 设置界面布局优化

**之前**：
- API 地址（单独入口）
- 模型名称（单独入口）
- API 密钥（单独入口）
- 自定义提示词（单独入口）
- 重置配置

**现在**：
- **API 配置**（统一入口）- 包含 API 地址、模型名称、API 密钥
- **提示词配置**（单独入口）- 自定义 AI 分析提示词
- **重置配置**

### 2. API 统一配置对话框

点击「API 配置」后弹出的对话框包含：


### 3. 功能特性

#### API 配置对话框

- **统一编辑** - 三个配置项在一个页面编辑，方便快捷
- **URL 验证** - 自动验证 URL 格式（必须以 http:// 或 https:// 开头）
- **批量保存** - 点击「保存」按钮一次性保存所有配置
- **快速重置** - 对话框内提供「重置」按钮，快速恢复默认值
- **智能提示** - 每个输入框都有默认值提示

#### 提示词配置

- **独立配置** - 提示词配置保持独立入口
- **多行编辑** - 支持多行文本编辑，方便编写复杂提示词
- **即时保存** - 点击保存后立即生效

## 用户体验改进

### 优势

1. **更直观** - API 相关配置集中管理，逻辑更清晰
2. **更高效** - 无需多次打开对话框，一次完成所有 API 配置
3. **更易维护** - 相关配置项放在一起，方便统一管理
4. **更灵活** - 提示词独立配置，方便单独调整

### 使用流程

#### 配置 API

1. 打开插件设置
2. 找到「AI 代码分析配置」分组
3. 点击「API 配置」
4. 在弹出的对话框中填写：
   - API 地址
   - 模型名称
   - API 密钥
5. 点击「保存」按钮

#### 配置提示词

1. 打开插件设置
2. 找到「AI 代码分析配置」分组
3. 点击「提示词配置」
4. 在多行编辑框中输入自定义提示词
5. 点击「保存」按钮

#### 重置配置

**方式一**：在 API 配置对话框中
1. 点击「API 配置」
2. 点击「重置」按钮
3. API 地址、模型名称、API 密钥恢复为默认值

**方式二**：在设置主界面
1. 点击「重置配置」
2. 所有配置（包括提示词）恢复为默认值

## 技术实现

### 代码结构

```java
// 设置入口
builder.addText("API 配置")
       .summary("配置 API 地址、模型名称和 API 密钥")
       .onClick((ui, preference) -> {
           showApiConfigDialog(ui, context);
       });

builder.addText("提示词配置")
       .summary("自定义 AI 分析提示词")
       .onClick((ui, preference) -> {
           showMultilineInputDialog(ui, context, ...);
       });

// API 统一配置对话框
private void showApiConfigDialog(PluginUI ui, PluginContext context) {
    // 构建包含三个输入框的视图
    PluginView view = ui.buildVerticalLayout()
        .addTextView().text("API 地址")
        .addEditText("api_url")
        .addTextView().text("模型名称")
        .addEditText("model_name")
        .addTextView().text("API 密钥")
        .addEditText("api_key")
        .build();
    
    // 显示对话框
    ui.buildDialog()
        .setTitle("API 配置")
        .setView(view)
        .setPositiveButton("保存", ...)
        .setNeutralButton("重置", ...)
        .show();
}
```
