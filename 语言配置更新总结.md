## 🔧 技术实现

### 命名规范
- `strings.mtl` - 默认语言（中文）
- `strings-区域代码.mtl` - 其他语言

### 添加新语言步骤

1. 在 `assets` 目录创建新文件
   - 例如：`strings-ja.mtl`（日文）
   - 例如：`strings-ko.mtl`（韩文）

2. 复制 `strings.mtl` 的内容

3. 将所有中文翻译为目标语言

4. 保持键名（冒号前的部分）不变

### 示例：添加日文支持

```
# strings-ja.mtl
plugin_name: 開発者ツールプラグイン
plugin_description: 強力な開発者ツールプラグイン
plugin_author: カン・ガー
features_title: 機能説明
encode_decode_function: エンコード/デコード
...（其他翻译）
```

```java
// 方式1：使用花括号（推荐用于 UI 构建）
```

## ⚠️ 注意事项

1. **键名格式**：只能使用字母、数字、下划线和点
2. **换行符**：使用 `\n` 表示换行
3. **续行符**：使用 `\` 在行末可续行
4. **注释**：使用 `#` 开头的行为注释
5. **特殊字符**：需要使用转义字符
